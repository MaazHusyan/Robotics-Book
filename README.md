# Physical and Humanoid Robotics: From Fundamentals to Advanced Applications

A comprehensive educational resource covering the theory and practice of robotics, with special focus on humanoid robots and real-world applications.

## üìö Book Overview

This book provides a complete learning path from basic robotics concepts to advanced humanoid robot design and implementation. It's designed for students, educators, makers, and professionals who want to understand both the theoretical foundations and practical applications of modern robotics.

### üéØ Learning Objectives

- Master fundamental robotics concepts including kinematics, dynamics, and control theory
- Understand humanoid robot design principles and bipedal locomotion
- Learn perception systems and AI integration in robotics
- Explore real-world case studies from leading robotics companies
- Gain hands-on experience with simulation tools and ROS/ROS2

## üöÄ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/download/) version 20.0 or above
- Basic understanding of physics and programming concepts

### Installation

```bash
npm install
```

### Local Development

```bash
npm start
```

This command starts a local development server and opens up a browser window at `http://localhost:3000`. Most changes are reflected live without having to restart the server.

### Build

```bash
npm build
```

This command generates static content into the `build` directory and can be served using any static contents hosting service.

## üåê Deployment

Using SSH:

```bash
USE_SSH=true npm deploy
```

Not using SSH:

```bash
GIT_USER=<Your GitHub username> npm deploy
```

If you are using GitHub pages for hosting, this command is a convenient way to build the website and push to the `gh-pages` branch.
